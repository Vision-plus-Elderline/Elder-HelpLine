# Elder Line Background Setup

## Instructions

To complete the login page background setup, you need to:

1. **Save the Elder Line background image**:
   - Take the Elder Line image you provided (the one with the elderly woman and the "National Helpline for Senior Citizens" text)
   - Save it as `elder-line-bg.jpg` in the `public` folder of your project
   - The file path should be: `public/elder-line-bg.jpg`

2. **The Auth page has been updated** with:
   - Background image support using the saved image
   - Semi-transparent overlay for better text readability
   - Enhanced styling with backdrop blur effects
   - White text for the logo and title for better contrast
   - Translucent cards with backdrop blur for a modern look
